Bugs Found & Fixed

02
Doubled nav labels — "Home Home / About About" Visual
Fix

Viora's navbar uses two <span class="text" data-splitting> per link for a JS character-animation hover effect. Without splitting.js initialising, both spans rendered simultaneously. Removed the duplicate spans — each nav link now has a single plain text anchor. class="text-menu" added to all <li> elements (required for Viora nav link CSS rules).

Header links invisible on first load (before scroll) CSS
Fix

Root cause: Viora's .header.header-fixed starts with transform: translateY(-100%); visibility: hidden;. JS only adds .is-fixed (revealing the header) after 300px of scroll. Override added to bejundas.css: transform: none; visibility: visible; pointer-events: all; — permanently visible on all pages.

Footer completely invisible — white text on white background CSS Variable
Fix

Footer uses background-color: var(--Secondary) where --Secondary: #154040 is defined in corporate.css, not in global styles.css. Without that file loaded the footer background was transparent. Fix: created hub/base_hub.html which loads corporate.css via a {% block demo_css %} block, and updated all hub templates to extend this instead of core/base.html directly.

{% extends %} not first tag — Django TemplateSyntaxError Django
Fix

All 7 hub templates were initially written with {% load static %} on line 1 and {% extends %} on line 2. Django requires {% extends %} to be the absolute first tag in any template. Fixed by swapping tag order in all affected files.

service.name referenced — field is service.title Template
Fix

Home template and footer both referenced {{ service.name }} but the Service model field is title. Fixed in both home.html and footer.html by replacing with {{ service.title }}.

GitHub push blocked — Mapbox API token in map.js Secret Scan
Fix

GitHub push protection detected a Mapbox demo token in static/viora/js/map.js. Used git reset --soft HEAD~1 + git restore --staged to remove both map files from the commit without losing other changes. Files added to .gitignore. Recommitted and pushed cleanly.
